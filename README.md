# Ejecutar en desarrollo

1. Clonar el Repositorio
2. Ejecutar

```
yarn install
```

3. Tener NestJs instalado

```
npm i -g @nest/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```

5. Reconstruir BD

```
http://slocalhost:3000/api/v2/seed
```

### Stack Utilizado

- MongoDB
- Nest
